Help What functions are available to be called by the Lua script

Say

Make mobs say a message using obj.Say() 
obj.Say("Hello World", 250, room, player)

param 1: text to send to the player
param 2: delay to send in ms
param 3: room object
param 4: player object

Output:
Hello world
If you want to say bob says "hello" you would need to do this: 
Example:
obj.Say(obj.getName(mob) .. "says Hello", 250, room, player)

Output:
Bob says Hello

Get Name

Returns the name of the player or mob, based on the parameter passed in. 
obj.getName(player)

param 1: mob or player object

Output:
Bob
